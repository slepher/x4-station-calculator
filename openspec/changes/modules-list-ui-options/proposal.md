# Proposal: Module List UI Options Enhancement

## Intent
修改模块列表的若干ui选项

## Scope
  - 用户规划区标题增加1x, 2x, 3x, 5x, 10x 选项, 用于调整规划模块的数量
  - 将计算劳动力加成选项移动到自动工业区标题栏
  - 自动工业区增加偏好种族选择下拉列表, 内容从consumption中获取
  - 将计算补给区劳动力加成选项移动到自动补给区标题栏 

## Approach
Follow existing Vue 3 + TypeScript patterns, use Tailwind CSS for styling, and ensure all changes are consistent with the current codebase architecture.
