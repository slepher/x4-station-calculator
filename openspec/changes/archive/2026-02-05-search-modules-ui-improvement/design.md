# 模块搜索UI改进设计

## 技术实现
优化搜索组件的界面布局。

## 修改内容
- 修改搜索框位置到ModuleList顶部
- 修改搜索框方向向右方弹出
- 取消搜索结果预览区, 点击弹出的模块之后直接添加模块到规划模块区, 表现和之前点击预览区的添加按钮一致
- 点击弹出的模块之后, 不关闭弹出菜单, 继续保持弹出菜单的显示状态, 用户点击菜单外区域, 关闭弹出框(确认之前是否存在这个逻辑, 如果已存在就不用改)
- 添加模块之后, 在新添加的模块上面加一个动画效果, 和hover效果一致, 持续适当的时间(比如0.3s)
- 当用户快速添加多个模块时, 每个模块的动画应该独立显示, 按照各自的时序开始和结束, 所有动画应该同时可见且互不干扰
- 更新模块数量时, 该模块也应该显示动画效果
- 模块的数字部分要有独立的闪烁动画效果, 区别于整体模块的边框动画

## 注意事项
保持与现有功能的兼容性。
