# Resource Dashboard 测试套件

此目录包含与 Resource Dashboard 相关的所有测试，根据功能模块进行了组织。

## 目录结构

- `volume-analysis/` - 体积分析功能相关测试
- `storage-planning/` - 存储规划功能相关测试
- `miscellaneous/` - 其他杂项功能测试（如人员配置等）
- 根目录 - 统一资源仪表盘核心功能测试

## 文件命名约定

测试文件按任务编号命名（task3-, task4-, etc.），便于追踪开发进度和功能迭代。

## 测试覆盖范围

- 统一资源仪表盘组件合并
- 视图模式切换（数量、体积、经济）
- 体积分析功能
- 存储规划功能
- 经济视图分组功能
- 资源视图分组功能
- 统一排序逻辑
